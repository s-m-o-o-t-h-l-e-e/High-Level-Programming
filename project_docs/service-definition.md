# 서비스 정의서

## 목적

이 프로젝트는 국내 유가 변동을 단순 시계열이 아니라 국제 유가, 환율, 뉴스/지정학 리스크와 함께 분석하는 것을 목표로 합니다.

## 분석 대상

- 국내 휘발유 평균 가격
- 국제 유가 WTI/Brent
- 원/달러 환율
- 이란, 미국, 호르무즈, 원유 관련 뉴스 리스크
- 지정학적 이벤트 리스크

## 사용자 흐름

1. 로컬 서버 실행
2. `/docs` 실행 문서에서 필요한 API 선택
3. `/docs`에서 API 테스트
4. 필요 시 `/refresh`로 최신 데이터 갱신

## 주요 화면

- API Docs: 필요한 API를 선택하고 바로 실행하는 커스텀 문서 화면
- 7일 예측 테이블: 예측 유가와 뉴스/환율 기반 변화 이유
- 전체 그래프 갤러리: EDA, 히트맵, 산점도, 히스토그램, 유가 추이

## 서버 책임

`oil_forecast_service/api`는 다음 책임을 갖습니다.

- 온라인 데이터 수집
- 전처리 및 스케일링
- EDA/가설 검증
- LSTM 예측
- 그래프 저장
- API 제공

`oil_forecast_service/web`은 웹 화면을 담당합니다.

- 커스텀 API Docs 화면
- 그래프 갤러리 화면
- 공통 CSS와 JavaScript

## 결과물 책임

`oil_forecast_service/outputs`는 서버와 웹 화면에서 바로 사용하는 최신 분석 산출물을 보관합니다.

- CSV 분석 결과
- 평가 요약
- PNG 그래프
