# 서비스 정의서

## 목적

이 프로젝트는 국내 유가 변동을 단순 시계열이 아니라 국제 유가, 환율, 뉴스/지정학 리스크와 함께 분석하는 것을 목표로 합니다.

## 분석 대상

- 국내 휘발유 평균 가격
- 국제 유가 WTI/Brent
- 원/달러 환율
- 이란, 미국, 호르무즈, 원유 관련 뉴스 리스크
- 지정학적 이벤트 리스크

## 사용자 흐름

1. 로컬 서버 실행
2. 홈페이지에서 최신 지표와 그래프 확인
3. `/docs`에서 API 테스트
4. `/agent`에 질문하여 현재 데이터 기반 해석 확인
5. 필요 시 `/refresh`로 최신 데이터 갱신

## 주요 화면

- 홈 대시보드: 최신 국내 유가, WTI/Brent, 환율, 뉴스 리스크 카드
- 7일 예측 테이블: 예측 유가와 뉴스 보정률
- 전체 그래프 갤러리: EDA, 히트맵, 산점도, 히스토그램, 유가 추이
- API Docs: FastAPI Swagger UI

## 서버 책임

`backend/api`는 다음 책임을 갖습니다.

- 온라인 데이터 수집
- 전처리 및 스케일링
- EDA/가설 검증
- LSTM 예측
- 그래프 저장
- 웹 대시보드 및 API 제공

## 결과물 책임

`evaluation/oil-price`는 보고서/발표에 사용할 정리된 산출물을 보관합니다.

- CSV 분석 결과
- 평가 요약
- PNG 그래프
- 발표 및 보고서 재사용 자료
